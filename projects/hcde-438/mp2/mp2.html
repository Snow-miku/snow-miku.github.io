Issue Deep-Dive

Issue: To maintain a consistant look, I create a JSON file for color palette. However, I struggle to save the content to javascript object and use the variable globaly.

Solviong Process: I tried the fetch chain and XMLHttpRequest() as I remembered from previous web programming courses. However, I could not have the returned json object as a global variable and a lot of times it shows undefined because the request is always slower than the assigning. I also tried .require and aync/await functions but neither worked. Eventually, I figured out that p5.js has its own preload() function for me to load JSON files before anything else is loaded.
