The fp0 Writeup

Last updated: 2023/2/28

Maou To Yuusha

This game is a battle game that allow user to pick a character and beat enemy in a turn-based strategic game. The character can level up and the enemy will increase in difficulty as well.

Two Versions of the Project

Prototype version

The prototype version will have a complete menu interface where the user can set up their character. It will use the firebase for storing character data such as the look, level, and damage. This version will also have the entire functionality for the game control mechanics that it will shift face when the user click start game, increase the difficulty of the enemy as the player progresses, and choices for the player to either attack, save the game, or defence. The enemy will also be implemented to attack back after every player attack.

Final version

It will allow user to switch to different character with an input of character name if it is fetchable. Also, the music, background picture, and the styling of the interface will be further enhanced for aesthetics. Moreover, a more profound calculation of the attack damage will be implemented so that user can view character's attack, defence, and hp statistics.

Development process

  1. STPE 1: Create new repository for the prototype, and tranplant mp3 into the new github repository.

  2. STPE 2: Initialize the `package.json` with `npm init`. Install a local development server, a build tool, a deployment tool, and any library necessary. Update the package.json script accordingly.

  3. STPE 3: Implement the new interface, draw all the canvas elements.

  4. STPE 4: Add functionality for the attack to be turn-based, and the enemy is able to fight back. Also store the difficulty level, so the enemy will gain higher attack damage and hp after every player win.

  5. MILESTONE: PROTOTYPE VERSION.

  6. STPE 5: Update the music and background image in different phase of the application.

  7. STPE 6: Add funtions to query API for characters' name and image, and allow user to change it in settings/characters.

  8. STPE 7: Implement cross-platform accessibility so the game works in different devices.

  9. STPE 8: Debugs, tests, and polishes the game.

  10. MILESTONE: FINAL VERSION.

Known unknowns
